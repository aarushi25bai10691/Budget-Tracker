# Problem Statement & Scope

## 1. Problem Statement
Many individuals struggle to manage and track their personal spending, often losing sight of where their money goes each day. Manual or scattered tracking methods can lead to poor financial decisions and missed savings opportunities. There is a need for an accessible, user-friendly tool to record, categorize, and summarize expensesâ€”helping users improve awareness and control over their budgets.

## 2. Scope of the Project
This project is a **Command-Line Budget Tracker** that:
* Allows users to record expenses with date, category, amount, and description.
* Stores expense data in a text file for long-term accessibility.
* Displays a summarized report showing total spending per category.
* Validates input for categories and expense amounts.
* Is designed for modular extensibility and easy customization.

## 3. Target Users
* Students and individuals wishing to practice foundational programming skills.
* Anyone seeking to organize and analyze daily personal spending.
* Beginners interested in Python, file handling, and basic data processing.

## 4. High-Level Features
* **Interactive CLI:** Menu-driven prompts for recording and viewing expenses.
* **Modular Design:** Separate modules for expense management, summary reporting, and validation.
* **Input Validation:** Ensures categorical and numerical integrity of user input.
* **Persistent Storage:** Expenses are saved in a plain text file for future reference.
* **Summary Reporting:** Instantly summarizes total spending by category for analysis.
